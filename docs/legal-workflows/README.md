# Legal Workflow Showroom

This folder contains user-facing workflow dossiers for the Forge Legal Department. Each dossier should make a workflow understandable to a professional who has not seen the implementation.

## Standard dossier structure

- Benefits brief
- Demo guide and video talk track
- Repeatable test plan
- Synthetic test documents
- Optional briefing deck

## Standard quality bar

Each workflow should answer three questions clearly:

1. **What does it do?**
2. **Why is it valuable?**
3. **How should a professional evaluate it?**
