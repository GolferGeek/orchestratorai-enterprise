# Document Onboarding Test Documents

These files are synthetic demonstration documents for testing the Document Onboarding workflow. They are not legal advice, do not reflect real client facts, and should not be reused as production legal forms.

## How to use

- Upload one file to test single-document classification and specialist routing.
- Upload all three files together to test mixed-packet handling.
- Compare the workflow output to `../test-plan.md`.

## Expected specialist coverage

- `scenario-01-saas-msa-high-risk.md` should trigger contract, privacy, IP, compliance, and possibly corporate review.
- `scenario-02-separation-agreement.md` should trigger employment, litigation, contract, and compliance review.
- `scenario-03-commercial-lease.md` should trigger real-estate, contract, compliance, and litigation review.
