## Benefits

- **Rehearse hostile questioning before the real deposition or hearing.** The simulator puts a witness through an adaptive cross-exam so weak answers, evasiveness, and credibility problems surface in practice rather than on the record.

- **Coach from specific performance, not vague impressions.** The run produces a transcript and debrief so attorneys can point to concrete weak moments and improve them.

- **Target the themes that matter most.** Focus the simulation on timeline gaps, prior-statement issues, factual weaknesses, or another high-risk topic and pressure-test it directly.

- **Build repetition into witness prep.** Repeat runs make it practical to compare witness performance across different topics and coaching rounds.

- **Keep rehearsal materials private.** Witness answers, transcripts, and debrief output remain inside your environment.

## Features

- Interactive turn-by-turn cross-examination
- Adjustable question count and optional thematic focus
- Real-time witness answer entry with adaptive follow-up questioning
- Transcript capture for every session
- Post-run debrief covering weak moments, patterns, and coaching guidance
- Repeat-run workflow for iterative practice
- Shared matter context with the deposition preparation workspace

## When to use it

- You want a live witness rehearsal before deposition, hearing, or testimony
- You need to test how a witness handles aggressive follow-up questioning
- You want a coaching artifact after a prep session
- You need to compare performance across multiple practice runs

## How it works

1. Open the **Simulation** tab in the deposition preparation workspace
2. Configure the question count and optional focus area
3. Start the simulation and answer each question as the witness would
4. Review the transcript and debrief after the run completes
5. Repeat the simulation to improve weak answers or test a new theme
