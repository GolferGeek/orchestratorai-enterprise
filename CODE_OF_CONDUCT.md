# Code of Conduct

This project should be a professional and respectful space for technical discussion.

## Expected Behavior

- Be direct, constructive, and respectful.
- Focus feedback on code, design, documentation, and outcomes.
- Assume good intent, but ask for clarification when something is ambiguous.
- Keep discussions relevant to the project.

## Unacceptable Behavior

- Harassment, discrimination, personal attacks, or sustained disruption.
- Publishing private information, credentials, or security-sensitive details.
- Intentionally misleading reports, abusive comments, or spam.

## Reporting

If you see behavior that makes the project unsafe or unproductive, contact the repository owner with a short description and any relevant links or context.
